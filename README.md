# Голосовой ввод для Space Station 14 (и не только) (на базе Faster-Whisper)

Скрипт позволяет наговаривать текст в микрофон по нажатию клавиши **F4**, автоматически распознаёт живую речь и моментально вставляет готовый текст в чат игры.

## 🛠 Инструкция по установке

1. Установите [Python 3.9 или новее](https://www.python.org/). При установке обязательно поставьте галочку **"Add Python to PATH"**.
2. Скачайте этот репозиторий себе на ПК (Кнопка **Code** -> **Download ZIP** вверху страницы).
3. Распакуйте архив в любую папку.
4. Запустите файл **`libraries.bat`** один раз, чтобы автоматически скачались все необходимые библиотеки. После успешной установки закройте консоль.

## 🚀 Запуск и использование

1. Запустите сам скрипт **`ss14_voice.py`** двойным кликом (или через консоль: `python ss14_voice.py`).
2. Зайдите в игру, откройте чат, нажмите на строке и удерживайте клавишу **F4**.
3. Наговорите фразу и отпустите кнопку. Текст автоматически появится в чате!

---

## ⚙️ Важные настройки и нюансы

* ⚠️ **Точность распознавания:** Нейросеть работает **не идеально**. Если у вас смазанная дикция, шум на фоне или ИИ впервые слышит редкое слово, он может ошибиться в падеже или букве. Будьте готовы иногда быстро подправлять текст руками в чате перед отправкой.
* 📝 **Кастомизация словаря:** Вы можете добавлять новые термины и сленг в файл `ss14_words.txt` через запятую.
* ⌨️ **Смена кнопки активации:** По умолчанию стоит **F4**. Если вы хотите изменить её (например, на `Caps Lock` или `F2`), откройте файл `ss14_voice.py` через обычный Блокнот, найдите в самом верху вот эту строчку и впишите туда свою кнопку:

  HOTKEY = "f4"
